🎨 FEATURES INCLUDED

✨ Main Screen
  → Animated gradient background (breathing effect)
  → Floating particle animations (14 ambient particles)
  → Glowing orb button (center, pulsing, interactive)
  → "Shake for your fortune" hint text
  → Rarity badge (glowing, dynamic color)

✨ Interaction
  → Tap the orb on any platform
  → Shake on iOS/Android (accelerometer detection)
  → Haptic vibration feedback (mobile only)
  → 800ms debounce to prevent rapid fires

✨ Fortune Pull Experience
  1. User shakes/taps
  2. Phone vibrates (haptics)
  3. Rarity is selected (probabilistic: 70% Common, 20% Rare, 8% Epic, 2% Legendary)
  4. Sound effects play (sparkle → rarity chime → reveal)
  5. Quote is fetched from ZenQuotes API
  6. Card animates upward with FadeInUp
  7. Rarity badge glows with unique color

✨ Sound & Audio
  → Sparkle effect (magic spell sound)
  → Reveal chime (fairy bells sound)
  → Different rarity sounds (arcade, fantasy, epic)
  → All free from Mixkit API

✨ Visual Polish
  → Smooth 60fps animations (React Native Reanimated)
  → Glassmorphic card styling
  → Neon glow effects
  → Soft shadows and blur
  → Dark mode theme (deep purple, neon blue, soft pink)
  → Responsive layout
